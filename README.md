# 3DS BAX Animations

A collection of BAX animations for the 3DS. (Only one so far...)

## GRUB

A BAX animation that mimics the look of the GRUB menu found in Linux.
Note: On the 3DS, if you look hard enough, you can see some artifacting. I, for the love of all that is holy, cannot remedy this. (Yes, I built the `.bax` with the option with the least compression.)
